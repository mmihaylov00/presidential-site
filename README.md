# Presidential Campaign Landing Prototype

Static prototype with three visual versions:

- `index.html` - calm statesman direction
- `movement.html` - modern movement direction
- `people.html` - citizen-first regional direction

The volunteer forms are local mocks only. They validate fields and show success/error states, but do not send, store, or persist personal data.

Open any HTML file directly in a browser. No local server is required.
